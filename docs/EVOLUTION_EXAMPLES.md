## Why Evolution Matters

### 1. Visual Reputation
Users can instantly see badge/domain status. High-level badges signal:
- Experience (time invested)
- Capabilities (features unlocked)
- Activity (achievements earned)
- Trustworthiness (provable history)

### 2. Status Symbol
Like limited edition sneakers or verified Twitter accounts:
- Level 10+ badges are rare
- Elite users showcase their evolved NFTs
- Social proof for agent marketplaces
- Collectible value increases with evolution

### 3. Economic Value
Evolution drives secondary market:
- Level 0 badge: $5-10
- Level 5 badge: $20-50
- Level 10 badge: $100-500
- Level 12+ badge: $1000+

Buyers pay premium for evolved badges with proven history.

## Implementation Details

### SVG Rendering
Each evolution level triggers different visual parameters:

```javascript
// Pseudocode
function renderNFT(level, fragments) {
  const borderWidth = 2 + (level * 0.5); // 2px ‚Üí 8px
  const glowColor = level < 5 ? 'cyan' : level < 10 ? 'purple' : 'gold';
  const glowIntensity = level * 2; // 0px ‚Üí 24px
  
  return `
    <svg>
      <rect 
        stroke="${glowColor}" 
        stroke-width="${borderWidth}"
        filter="drop-shadow(0 0 ${glowIntensity}px ${glowColor})"
      />
      ${renderFragments(fragments.slice(0, 15))}
    </svg>
  `;
}
```

### Fragment Display Logic
- Max 15 fragments shown on image
- Ordered by earnedAt timestamp (oldest first)
- If >15 total, show "+ X more" indicator
- Fragment shapes: ‚¨§ (domain), ‚¨¢ (badge), ‚≠ê (special)

**"Evolution creates a progression system like video games, but with real economic value."**
